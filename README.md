## Aplikacja stworzona podczas nauki Spring Framework 5.

### Przegląd technologi użytych podczas nauki:

- Spring Framework 5
- Spring Boot 2
- Spring Data JPA
- Spring MVC
- Spring MockMVC
- Spring WebFlux
- Spring Web Client
- Spring Data MongoDB
- Hibernate
- Project Lombok
- MapStruct
- Maven
- Gradle
- JUnit 4 and JUnit 5
- Mockito
